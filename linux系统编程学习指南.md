# Linux系统编程学习指南

## 学习误区与正确方向

很多同学都说Linux难学，那是因为走错了方向。零基础就去点亮一颗LED灯很容易劝退。

**正确学习方法：**
- 学习Linux必须从应用层开始
- 不用开始着急买开发板，一台电脑就行
- 重点学习系统编程，而不是驱动开发

## 第一阶段：基础命令和工具

### Linux常用命令
**注意：** Linux命令有很多，千万不要都去学一遍，重点掌握常用的即可。

**必须掌握的命令：**
- `ls` - 列出目录内容
- `cd` - 切换目录  
- `cp` - 复制文件
- `mv` - 移动文件
- `tar` - 打包压缩

**学习目标：** 能搞清楚路径，找到地方写代码即可。

### 开发工具
- **Vim** - 必须会用（如果实在不习惯可以用VSCode）
- **GCC** - C语言编译器

**学习建议：** 花几天时间熟悉一下，后面天天在Linux下做开发，用多了就熟练了。看到有些同学花几个月去学习命令，真的没有必要。

## 第二阶段：系统编程核心内容

### 第一块：文件系统IO和标准IO

#### 文件系统IO（底层操作）
**必须掌握的函数：**
- `open` - 打开文件
- `read` - 读取文件
- `write` - 写入文件  
- `lseek` - 文件定位
- `close` - 关闭文件
- `mmap` - 内存映射

#### 标准IO操作（高级操作）
**必须掌握的函数：**
- `fopen` - 打开文件流
- `fread` - 读取数据
- `fwrite` - 写入数据
- `fseek` - 文件流定位

**学习重点：**
- 学会看man手册
- 知道各个参数的意义

### 第二块：进程管理

**核心概念（面试重点）：**
- 进程调度
- 进程状态
- 僵尸进程
- 死锁

**必须掌握的函数：**
- `fork` - **必须重点掌握**
- 进程回收：`wait`/`waitpid`
- `exec`函数族

### 第三块：进程间通信（IPC）

**常见的进程间通信方式：**
- 无名管道（Pipe）
- 有名管道（FIFO）
- 信号（Signal）
- 消息队列（Message Queue）
- 共享内存（Shared Memory）
- 进程同步信号量（Semaphore）

### 第四块：多线程编程

**线程操作函数：**
- `pthread_create` - 线程创建
- `pthread_join` - 线程回收
- `pthread_exit` - 线程退出

**线程同步机制：**
- 互斥锁（Mutex）
- 条件变量（Condition Variable）
- 线程池（Thread Pool）

**应用场景：** 如果企业涉及高并发业务，这也是面试重点内容。

### 第五块：网络编程

#### TCP服务器
**核心函数：**
- `socket` - 创建套接字
- `bind` - 绑定地址
- `listen` - 监听连接
- `accept` - 接受连接
- `recv` - 接收数据

#### TCP客户端
**核心函数：**
- `socket` - 创建套接字
- `connect` - 连接服务器
- `send` - 发送数据

#### UDP协议
**核心函数：**
- `recvfrom` - 接收数据
- `sendto` - 发送数据

#### 并发服务器模型
- 多线程（Multi-threading）
- `select` - I/O多路复用
- `epoll` - 高性能I/O多路复用

## 学习总结

### 难度评估
Linux系统编程真的不难，比STM32要简单得多。只要C语言基础过硬，一个月就能搞定。

### 就业建议
至于点灯（Linux驱动开发），对于着急就业、基础一般的同学就不要学了，好好准备应用层，足够找到一份工作。

## 附录：开发工具配置

### Ctags标签生成

```bash
# 生成系统头文件标签
cd /usr/include
sudo ctags -R --c-kinds=+p --fields=+iaS --extra=+q .
```

### Vim配置

将生成的tags加入vim搜索路径：
```vim
" 在~/.vimrc中添加
set tags+=/usr/include/tags
```

### Tags功能验证

**方式1：**
```vim
:tag FILE
```

**方式2：**
```bash
vim -t FILE
```

### Tags工具关键快捷键

- `Ctrl + ]` - 跳转至符号定义处
- `Ctrl + o` - 回退到跳转前位置
- `:tag 符号` - 跳转至指定符号的定义

### 常用文件操作函数参考
- `fopen` - 打开文件
- `fgetc` - 读取字符
- `fclose` - 关闭文件
- `fputc` - 写入字符
