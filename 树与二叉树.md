# 树与二叉树

**学习日期：** 2025年12月1日  
**学习目标：** 掌握树和二叉树的基本概念、核心性质和应用场景

## 1. 树的基本概念

### 1.1 树的定义
树是n(n≥0)个结点的有限集。当n=0时，称为空树。在任意一棵非空树中：
- 有且仅有一个特定的称为**根**的结点
- 当n>1时，其余结点可分为m(m>0)个互不相交的有限集，每个集合本身又是一棵树，称为根的**子树**

### 1.2 基本术语

#### 结点相关术语
- **根结点**：树的最顶层结点，没有父结点
- **叶子结点**：度为0的结点，也称为终端结点
- **分支结点**：度不为0的结点，也称为非终端结点
- **父结点**：一个结点的直接前驱结点
- **子结点**：一个结点的直接后继结点
- **兄弟结点**：具有相同父结点的结点
- **祖先结点**：从根到该结点所经分支上的所有结点
- **子孙结点**：以某结点为根的子树中的所有结点

#### 树结构相关术语
- **结点的度**：结点拥有的子树数目
- **树的度**：树内各结点的度的最大值
- **结点的层次**：从根开始定义，根为第1层，根的孩子为第2层，依此类推
- **树的深度/高度**：树中结点的最大层次
- **有序树**：树中结点的各子树从左至右有次序
- **无序树**：树中结点的各子树无次序
- **森林**：m(m≥0)棵互不相交的树的集合

## 2. 二叉树

### 2.1 二叉树的定义
二叉树是n(n≥0)个结点的有限集合，该集合或者为空集，或者由一个根结点和两棵互不相交的、分别称为根结点的左子树和右子树的二叉树组成。

### 2.2 二叉树的特点
- 每个结点最多有两棵子树
- 子树有左右之分，次序不能任意颠倒
- 即使树中某结点只有一棵子树，也要区分它是左子树还是右子树

### 2.3 二叉树的性质

#### 性质1：在二叉树的第i层上至多有$2^{i-1}$个结点
- 第1层：$2^{0} = 1$个结点
- 第2层：$2^{1} = 2$个结点
- 第3层：$2^{2} = 4$个结点
- ...
- 第i层：$2^{i-1}$个结点

#### 性质2：深度为k的二叉树至多有$2^{k}-1$个结点
- 深度1：$2^{1}-1 = 1$个结点
- 深度2：$2^{2}-1 = 3$个结点
- 深度3：$2^{3}-1 = 7$个结点
- ...
- 深度k：$2^{k}-1$个结点

#### 性质3：对于任意二叉树，叶子结点数$n_0$和度为2的结点数$n_2$满足$n_0 = n_2 + 1$
- 证明思路：设度为1的结点数为$n_1$，总结点数$n = n_0 + n_1 + n_2$
- 总分支数$B = n - 1$（每个结点对应一个分支，除了根结点）
- 同时$B = n_1 + 2n_2$
- 因此$n - 1 = n_1 + 2n_2$
- 代入$n = n_0 + n_1 + n_2$得：$n_0 + n_1 + n_2 - 1 = n_1 + 2n_2$
- 化简得：$n_0 = n_2 + 1$

## 3. 特殊二叉树

### 3.1 满二叉树
- 定义：深度为k且有$2^{k}-1$个结点的二叉树
- 特点：
  - 每一层上的结点数都达到最大值
  - 叶子结点都在最底层
  - 编号规则：从上到下，从左到右

### 3.2 完全二叉树
- 定义：深度为k的，有n个结点的二叉树，当且仅当其每一个结点都与深度为k的满二叉树中编号从1到n的结点一一对应
- 特点：
  - 叶子结点只能出现在最下两层
  - 最下层的叶子结点都集中在树的左部
  - 如果有度为1的结点，只可能有一个，且该结点只有左孩子

### 3.3 满二叉树与完全二叉树的关系
- **满二叉树一定是完全二叉树**：因为满二叉树满足完全二叉树的定义
- **完全二叉树不一定是满二叉树**：当结点数小于$2^{k}-1$时，是完全二叉树但不是满二叉树

## 4. 二叉树的性质应用

### 4.1 计算问题示例

#### 示例1：给定深度求最大结点数
**问题：** 深度为5的二叉树最多有多少个结点？
**解答：** 根据性质2，深度为k的二叉树至多有$2^{k}-1$个结点
- 深度5：$2^{5}-1 = 32 - 1 = 31$个结点

#### 示例2：给定叶子结点数求度为2的结点数
**问题：** 一棵二叉树有10个叶子结点，求度为2的结点数？
**解答：** 根据性质3，$n_0 = n_2 + 1$
- $10 = n_2 + 1$ ⇒ $n_2 = 9$

#### 示例3：综合计算
**问题：** 一棵完全二叉树有1001个结点，其中叶子结点有多少个？
**解答：**
- 设叶子结点数为$n_0$，度为1的结点数为$n_1$，度为2的结点数为$n_2$
- 已知$n_0 + n_1 + n_2 = 1001$
- 根据性质3：$n_0 = n_2 + 1$ ⇒ $n_2 = n_0 - 1$
- 代入得：$n_0 + n_1 + (n_0 - 1) = 1001$ ⇒ $2n_0 + n_1 = 1002$
- 在完全二叉树中，$n_1$只能是0或1
- 如果$n_1 = 0$，则$2n_0 = 1002$ ⇒ $n_0 = 501$
- 如果$n_1 = 1$，则$2n_0 + 1 = 1002$ ⇒ $2n_0 = 1001$（不是整数，舍去）
- 因此叶子结点数为501个

### 5.3 快速计算练习
1. **深度为4的满二叉树有多少个结点？**
   - 答案：$2^{4}-1 = 15$个结点

2. **有20个叶子结点的二叉树，度为2的结点有多少个？**
   - 答案：$20 - 1 = 19$个

3. **深度为6的二叉树最多有多少个叶子结点？**
   - 答案：第6层最多有$2^{5} = 32$个叶子结点

4. **一棵二叉树度为2的结点有15个，度为1的结点有30个，求叶子结点数？**
   - 答案：$n_0 = n_2 + 1 = 15 + 1 = 16$个
