# 线性表 (Linear List)

## 1. 线性表的抽象数据类型 (ADT)

### ADT List 定义

```
ADT List {
    数据对象：D = {ai | ai∈ElemSet, i=1,2,...,n, n≥0}
    数据关系：R1 = {<ai-1, ai> | ai-1, ai∈D, i=2,3,...,n}
    基本操作：
        InitList(&L) //线性表的初始化
        DestroyList(&L)
        ClearList(&L)
        ListEmpty(L)
        ListLength(L)
        GetElem(L, i, &e)
        LocateElem(L, e,compare())
        PriorElem(L, cur_e, &pre_e)
        NextElem(L, cur_e, &next_e)
        ListInsert(&L, i, e)
        ListDelete(&L, i, &e)
        TraverseList(L, visit())
} ADT List
```

## 2. 基本操作详细说明

### 2.1 初始化与销毁
- **InitList(&L)** : 构造一个空的线性表L
- **DestroyList(&L)** : 销毁线性表L
- **ClearList(&L)** : 清空线性表L

### 2.2 状态查询
- **ListEmpty(L)** : 判断线性表L是否为空
- **ListLength(L)** : 返回线性表L的元素个数

### 2.3 元素操作
- **GetElem(L, i, &e)** : 返回线性表L中第i个位置的元素e
- **LocateElem(L, e, compare())** : 返回线性表L中第一个值为e的元素的位置
- **PriorElem(L, cur_e, &pre_e)** : 返回线性表L中元素cur_e的直接前驱
- **NextElem(L, cur_e, &next_e)** : 返回线性表L中元素cur_e的直接后继

### 2.4 修改操作
- **ListInsert(&L, i, e)** : 在线性表L中第i个位置插入元素e
- **ListDelete(&L, i, &e)** : 在线性表L中删除第i个位置的元素，并返回该元素
- **TraverseList(L, visit())** : 依次对线性表L中的每个元素调用visit()函数

## 3. 存储结构

### 3.1 顺序存储（顺序表）

#### 顺序表的特点：
- 以物理位置相邻表示逻辑关系
- 任何一个元素都可以任意访问（优点）

#### 顺序表元素的特点：
- 地址连续
- 依次存放
- 随机存取
- 类型相同

#### C语言实现示例：
```c
#define LIST_INIT_SIZE 100  // 初始化顺序表的分配量

typedef struct {
    ElemType *elems;  // 存储数组
    int length;                       // 当前长度
} SqList;

SqList L;
L.data=(ElemType *)malloc(LIST_INIT_SIZE * sizeof(ElemType));
```

### 3.2 链式存储（链表）

#### 链表的特点：
- 用指针表示逻辑关系
- 元素存储位置不连续
- 插入删除效率高
- 需要额外空间存储指针

#### 链表实现方式：
- 单链表
- 双向链表
- 循环链表
- 静态链表

---

**注意：** 以上函数是线性表的抽象接口定义，在C语言中需要根据具体的存储结构（顺序存储或链式存储）来实现这些函数。
